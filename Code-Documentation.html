This function will take an image and overlay it over an image map on mouse hover. You can only use one image, but it's a good way of bringing an edit function (on say, a list) to life on a mockup

Managing a left or top navigation in one place, and adding it programmatically to each mockup

It's a pain to have mockups that all use a similar navigation scheme but which can get out of date quickly because of changes in the navigation or top header area. This function will allow you to take a single image and add it across pages - thereby giving you control over many mockups by changing a single image. It works for center-aligned or left-aligned pages.

Example

Place this snippet inside the HEAD area of your page

			
			
			
			<script src="http://mockups/HASP/Product/js/jquery-1.8.2.min.js"></script> 
			<script type="text/javascript" src="http://mockups/HASP/Product/js/add-left-nav.js"></script>	
			
			<link rel="stylesheet" href="http://mockups/HASP/Product/css/gradient-background.css" type="text/css" />
